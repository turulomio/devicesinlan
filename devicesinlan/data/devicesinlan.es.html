NAME

devicesinlan: Busca todos los dispositivos de tu LAN. Después, puedes establecer alias a tus dispositivos conocidos para detectar dispositivos extraños en tu red.

SYNOPSIS

devicesinlan [-h] [--version] [--method {PingArp,ScapyArping,Scapy}] [--interface INTERFACE | --add | --remove | --list | --load LOAD | --save SAVE | --reset] [--debug DEBUG]

DESCRIPCIÓN

        Si lanzas DevicesInLan sin parámetros se ejecuta un asistente.

        Además puedes usar uno de estos parámetros.

        --interface

                Escanea la red del interfaz pasado como parámetro y lista los dispositivos detectados.

                Si un dispositivo no es conocido, se mostrará en rojo. Los dispositivos verdes son dispositivos confiables.

        --add

                Permite añadir un dispositivo conocido desde consola.

        --remove

                Permite borrar un dispositivo conocido desde consola.

        --list

                Muestra todos los dispositivos conocidos de la base de datos desde consola.

        --load

                Carga una lista de dispositivos conocidos en formato xml.

        --save

                Graba la lista de dispositivos conocidos a un fichero xml.

        --debug

                Devuelve información de depurado al ejecutar DevicesInLAN. Está desactivado por defecto

                Este parámetro puede tomar estas opciones: CRITICAL, ERROR, WARNING, INFO, DEBUG.

        --reset

                Borra todos los dispositivos conocidos.

MÉTODOS DE ESCANEADO

        DevicesInLan can use several methods to scan for devices. You just need to add the --method in console mode.

        PingArp

                It tries to make a socket connection with any device in the lan. Then it searches with 'arp' command the mac information

                El métoo PingArp es usado por defecto en las versiones de windows y linux.

        Scapy

                Usa Scapy para crear una peticion ARP y capturar su respuesta por cada ip en la subred, para conseguir la información de las MAC.

                This method can be used in windows and linux versions, but it needs to be executed with administrator role.

                Morever, you need to install 'npcap' in order to execute it in Windows.

        ScapyArping

                Uses Scapy arping function to create ARP request and capture its answer for each ip in the subnet to get macs information.

                This method can be used in windows and linux versions, but it needs to be executed with administrator role.

                Morever, you need to install 'npcap' in order to execute it in Windows.

EJEMPLOS

        devicesinlan

                Comando por defecto Usa el método PingArp.

        devicesinlan --method ScapyArping

                Usa el método ScapyArping.
